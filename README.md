# Sahil Modi Portfolio

A Jupyter Notebook-style portfolio built with Next.js, showcasing AI/ML projects, healthcare platforms, and full-stack development work through an interactive notebook-inspired interface.

## Overview

This portfolio presents a modern Jupyter Notebook-inspired design reflecting AI/ML development, full-stack engineering, and data-driven workflows. The interface includes:

* Notebook shell with interactive UI
* Code cells with syntax highlighting
* Output cells for project visualization
* Dark and light theme support
* Keyboard shortcuts for notebook navigation
* Interactive execution simulation
* Responsive notebook-inspired developer experience

## Tech Stack

* Next.js
* React.js
* TypeScript
* Tailwind CSS
* FastAPI
* Firebase
* React Syntax Highlighter

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.
